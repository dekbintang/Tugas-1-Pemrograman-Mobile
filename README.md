# HelloWorldApp

Aplikasi Android sederhana menggunakan Kotlin yang menampilkan Hello World dan identitas mahasiswa.

## Fitur

- Menampilkan teks Hello World
- Tombol untuk menampilkan Nama dan NIM
- Tombol berubah menjadi Sembunyikan setelah diklik

## Teknologi

- Kotlin
- Android Studio
- Min SDK API 24 (Android 7.0)
- Target SDK API 35

## Cara Menjalankan

1. Clone repository
```bash
git clone https://github.com/dekbintang/HelloWorldApp.git
```

2. Buka di Android Studio
3. Tunggu Gradle Sync selesai
4. Klik Run
